# 🎵 United States Top 50 Playlist Performance and Song Popularity Trend Analysis

## 📌 Project Overview

This project analyzes historical United States Top 50 playlist data to understand song popularity, ranking behavior, artist dominance, and content characteristics.

The project was developed for Atlantic Recording Corporation to support data-driven decisions regarding:

* Artist promotion strategies
* Release timing decisions
* Marketing spend optimization
* Catalog performance analysis

## ▶️ How to Run

Install packages:

```bash
pip install -r requirements.txt
```

Run analytics engine:

```bash
python main.py
```

Run dashboard:

```bash
streamlit run dashboard/app.py
```

Dashboard URL:

```text
http://localhost:8501
```
